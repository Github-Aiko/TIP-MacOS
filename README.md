# TIP-MacOS
 TIP For MacOS M1

## 1. Install Homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 2. Install Rosetta
```bash
softwareupdate --install-rosetta
```
